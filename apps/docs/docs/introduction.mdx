# Introduction

Welcome to Animate UI - a beautiful, modern UI component library with built-in animations.

## What is Animate UI?

Animate UI is a collection of beautifully designed, animated UI components for React applications. Built with TypeScript and powered by Framer Motion, it provides a set of ready-to-use components that bring life to your web applications through smooth, customizable animations.
