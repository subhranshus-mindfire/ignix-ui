Tab

Overview​

The Tabs component is a versatile navigation interface that allows users to switch between different views within the same context. Built with Framer Motion for smooth animations and powered by Class Variance Authority for flexible styling, it provides multiple variants and themes for different design needs.

Preview​

  • Preview
  • Code
import { Tabs } from '@mindfiredigital/ignix-ui';

function TabsDemo() {
  return (
    <Tabs
      options={["Home", "Profile", "Settings", "About"]}
      selected={0}
    />
  );
}

Installation​

  • npm
  • Yarn
  • pnpm
npx @mindfiredigital/ignix-ui add tab
yarn @mindfiredigital/ignix-ui add tab
pnpm @mindfiredigital/ignix-ui add tab

Usage​

Import the component:

import { Tabs } from '@mindfiredigital/ignix-ui';

Basic Usage​

function BasicTabs() {
  return (
    <Tabs
      options={["Tab 1", "Tab 2", "Tab 3"]}
      selected={0}
      value={(index) => console.log(`Selected tab: ${index}`)}
    />
  );
}

Interactive Example​

  • Preview
  • Code

Content area for the selected tab

function TabsWithContent() {
  const [activeTab, setActiveTab] = useState(0);
  
  const content = [
    "Account settings and preferences",
    "Security options and privacy",
    "Notification preferences",
    "Third-party integrations",
  ];

  return (
    <div className="space-y-4">
      <Tabs
        options={[
          "Account",
          "Security",
          "Notifications",
          "Integration"
        ]}
        variant="outline"
        size="md"
        value={setActiveTab}
      />
      <div className="mt-4 p-4 border rounded-lg">
        <p className="text-sm text-gray-600">
          {content[activeTab]}
        </p>
      </div>
    </div>
  );
}

Customization​

Custom Styling​

You can customize the appearance using Tailwind CSS classes:

<Tabs
  options={["Tab 1", "Tab 2", "Tab 3"]}
  className="bg-indigo-100 p-2 rounded-xl"
/>

Advanced Example with Content​

function TabsWithPanels() {
  const [activeTab, setActiveTab] = useState(0);

  const tabPanels = [
    {
      title: "Dashboard",
      content: (
        <div className="p-4">
          <h3 className="text-lg font-medium">Dashboard Overview</h3>
          <p className="mt-2 text-gray-600">Your dashboard content here</p>
        </div>
      ),
    },
    {
      title: "Analytics",
      content: (
        <div className="p-4">
          <h3 className="text-lg font-medium">Analytics Data</h3>
          <p className="mt-2 text-gray-600">Your analytics content here</p>
        </div>
      ),
    },
    {
      title: "Settings",
      content: (
        <div className="p-4">
          <h3 className="text-lg font-medium">Settings Panel</h3>
          <p className="mt-2 text-gray-600">Your settings content here</p>
        </div>
      ),
    },
  ];

  return (
    <div className="space-y-4">
      <Tabs
        options={tabPanels.map(tab => tab.title)}
        variant="gradient"
        theme="glass"
        value={setActiveTab}
      />
      <div className="mt-4 border rounded-lg overflow-hidden">
        {tabPanels[activeTab].content}
      </div>
    </div>
  );
}
