Code of Conduct

As contributors and maintainers of the Ignix UI package, we are committed to showing respect to all individuals who contribute to this project. This respect encompasses actions such as posting issues, updating documentation, submitting pull requests, providing constructive feedback in comments, and engaging in any other project-related activities.

In our communication through GitHub and other project channels, we are dedicated to maintaining a constructive and professional atmosphere. We will not tolerate personal attacks, trolling, public or private harassment, insults, or any other unprofessional conduct.

Our commitment extends to showing courtesy and respect to everyone involved in this project, regardless of their gender, gender identity, sexual orientation, disability, age, race, ethnicity, religion, or level of experience. We also hold the expectation that anyone contributing to the Ignix UI project will adhere to these principles.

In cases where a community member violates this code of conduct, the maintainers of the Ignix UI package may take appropriate actions, including the removal of issues, comments, and pull requests, or the blocking of user accounts.

If you experience or witness any behavior that goes against this code of conduct or if you have other concerns, please feel free to contact us via email.
