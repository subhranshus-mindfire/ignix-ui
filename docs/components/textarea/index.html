Textarea

Overview​

The Textarea component is an enhanced version of the standard HTML textarea with beautiful animations and interactive features. Built with Framer Motion, it provides various animation variants, auto-expanding capabilities, and interactive effects.

Preview​

  • Preview
  • Code
import { AnimatedTextarea } from '@mindfiredigital/ignix-ui';

function TextareaDemo() {
  const [value, setValue] = useState('');
  
  return (
    <AnimatedTextarea
      placeholder="Type something..."
      variant="clean"
      value={value}
      onChange={setValue}
    />
  );
}

Interactive Example​

  • Preview
  • Code
function InteractiveTextarea() {
  const [value, setValue] = useState('');
  const [variant, setVariant] = useState('clean');
  
  return (
    <div className="space-y-4">
      <select
        className="w-full p-2 border rounded"
        value={variant}
        onChange={(e) => setVariant(e.target.value)}
      >
        <option value="clean">Clean</option>
        <option value="expandable">Expandable</option>
        <option value="glowBorder">Glow Border</option>
        <option value="characterCount">Character Count</option>
        <option value="particleField">Particle Field</option>
        <option value="cosmic">Cosmic</option>
      </select>
      
      <AnimatedTextarea
        placeholder="Try different variants..."
        variant={variant}
        value={value}
        onChange={setValue}
      />
    </div>
  );
}

Props​

PropTypeDefaultDescription
placeholderstring-Placeholder text for the textarea
variantstring'clean'Animation variant to apply
valuestring-The controlled value of the textarea
onChange(value: string) => void-Callback when the value changes
textareaClassNamestring-Additional CSS classes for the textarea element
labelClassNamestring-Additional CSS classes for the label element

Available Variants​

  • Basic: 'clean', 'glowBorder', 'neonGlow'
  • Expansion: 'expandable', 'smoothExpand'
  • Features: 'characterCount', 'lineHighlight', 'syntaxHighlight'
  • Effects: 'particleField', 'wave', 'spotlight', 'liquid', 'cosmic', 'hologram'
  • Interactions: 'elastic', 'rippleEffect', 'gradientBorder'

Customization​

Custom Styling​

You can customize the appearance using CSS classes:

<AnimatedTextarea
  placeholder="Custom styled..."
  variant="clean"
  textareaClassName="border-purple-500 focus:border-purple-700"
  labelClassName="text-purple-600"
  value={value}
  onChange={setValue}
/>

Advanced Example with Markdown Preview​

function MarkdownEditor() {
  const [content, setContent] = useState('');
  const [preview, setPreview] = useState('');
  
  useEffect(() => {
    // Convert markdown to HTML (using your preferred library)
    const html = convertMarkdownToHtml(content);
    setPreview(html);
  }, [content]);

  return (
    <div className="grid grid-cols-2 gap-4">
      <AnimatedTextarea
        placeholder="Write markdown here..."
        variant="syntaxHighlight"
        value={content}
        onChange={setContent}
        textareaClassName="font-mono"
      />
      <div
        className="p-4 border rounded-md"
        dangerouslySetInnerHTML={{ __html: preview }}
      />
    </div>
  );
}
